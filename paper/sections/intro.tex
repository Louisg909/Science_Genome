Science Genome is scoped as a research framework for studying the flow and evolution of scientific ideas. The project objective is not paper discovery or citation search; it is analysis of idea lineage, inheritance, emergence, divergence, convergence, and author or stream contribution within scientific knowledge systems.

This artifact seals that scope by defining the framework in a form that can be read, evaluated, and applied without external project documents. It therefore serves as both publication-ready manuscript content and project-level completion record.

\subsection{Scope commitments captured by this artifact}
The framework specified here addresses the full in-scope objective set:
\begin{itemize}
    \item Representation of scientific artifacts and their relationships.
    \item Joint semantic and lineage analysis.
    \item Interpretable evidence for idea-flow hypotheses.
    \item Comparative analysis across branches, domains, and streams.
    \item Explicit verification and validation (V\&V).
    \item Reproducible, auditable research outputs.
\end{itemize}

\subsection{Out-of-scope protections}
To preserve scientific validity, this artifact explicitly excludes product expansion goals that are not required for core research questions. It does not claim causal proof of influence, does not replace expert interpretation, and does not bind the framework to any single implementation family.

\subsection{Role of this paper in project completion}
This paper is the final standalone artifact for the defined scope. It codifies:
\begin{itemize}
    \item the conceptual model,
    \item the formal estimation problem,
    \item the interpretation policy,
    \item the V\&V protocol,
    \item and the limitations required for responsible use.
\end{itemize}

Any compliant implementation can be evaluated against these definitions. In this way, the artifact completes the project scope at the research-framework level while remaining implementation-agnostic.
