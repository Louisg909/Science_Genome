This paper finalizes the Science Genome project scope as a standalone research artifact. It defines an implementation-agnostic framework for analyzing scientific idea flow through semantic inheritance decomposition on citation lineages, together with explicit verification, validation, interpretation, and limitation requirements.

The artifact is complete with respect to the scoped project objective: researchers can formulate idea-flow hypotheses, evaluate lineage and contribution structure, assess confidence and failure modes, and reproduce analytical outputs under auditable procedures.

Accordingly, this manuscript serves as the publishable sealing document for the project-level scope: a final framework specification that is interpretable, testable, and self-contained.
